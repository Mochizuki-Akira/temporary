気体・物質の状態 総合版

一、物質の三態

1. 状態変化

固体 ⇄ 液体 ⇄ 気体 の間で起こる状態変化：

  • 融解（固体 → 液体）／凝固（液体 → 固体）
  • 蒸発・沸騰（液体 → 気体）／凝縮（気体 → 液体）
  • 凝華（固体 → 気体）／昇華（気体 → 固体）

水の加熱による温度変化とエネルギー：

  • 融点付近：固体 → 固体と液体（融解；融解熱）
  • 液体の加熱：温度上昇
  • 沸点付近：液体 → 液体と気体（沸騰；蒸発熱）
  • 気体の加熱：温度上昇

2. 状態図：純物質の状態は温度と圧力で決まる

例：水 (H2O) の状態図

  • 蒸気圧曲線／融解曲線／昇華圧曲線
  • 三重点：固体・液体・気体が共存できる唯一の点
  • 超臨界状態：臨界点を超え液体と気体の区別がなくなる

比較：水と二酸化炭素の状態図

二、基礎の確認・気体の性質

1. 粒子の拡散と熱運動

・拡散：物質の粒子が自由に散らばって全体に広がる現象。

・熱運動：物質を構成する粒子が絶えず不規則に運動している。この運動を熱運動という。

2. 温度

・温度：熱運動の激しさに対応しているもの（単位：℃）。

※ 温度が高いほど、粒子の熱運動は激しい。

・分子の速度分布：

・絶対温度 T ：セルシウス温度 t に 273 を足したもの（単位：K）。

T［K］ = t［℃］ + 273

3. ボイルの法則（温度一定）

温度一定のとき、一定物質量の気体の体積 V は圧力 P に反比例。

例：
温度を一定に保ったまま、3.0×105 Paで10 Lの気体を15 Lの容器に入れると、圧力は何 Paになるか。

4. シャルルの法則（圧力一定）

圧力一定のとき、一定物質量の体積 V は絶対温度 T に比例。

例：
圧力を一定に保ったまま、27℃で6 Lの気体を −73℃ に冷却すると、体積は何 Lになるか。

5. ボイル・シャルルの法則

ボイル・シャルルの法則

PV/T = k3（一定）

P1V1/T1 = P2V2/T2

一定物質量の気体で、V は P に反比例し、T に比例。

例：
27℃・2.0×105 Paで 5.0 L を占める気体がある。
この気体を 87℃・1.0×105 Pa にすると、体積は何 Lになるか。

標準状態と気体定数 R

標準状態（0℃, 1.013×105 Pa）で 1 mol の体積 = 22.4 L。

気体定数 R（普遍定数）

  • R = 8.314 462 618… J·mol−1·K−1
  • 換算：R ≈ 0.082057 L·atm·mol−1·K−1 ≈ 8.314 kPa·L·mol−1·K−1

三、混合気体

1. 全圧と分圧

全圧：混合気体全体が示す圧力。

分圧：混合気体と同じ体積を占める各成分気体が示す圧力。

2. ドルトンの分圧の法則

混合気体の全圧は、各成分の分圧の和。

P = PA + PB

3. 分圧の計算

(1) 分圧と物質量

分圧の比は物質量の比に等しい。

PA / PB = nA / nB

(2) 分圧とモル分率

PA = ( nA / ( nA + nB ) ) × P

4. 混合気体の平均分子量

̄M = ( nA/(nA+nB) ) MA + ( nB/(nA+nB) ) MB

例：空気の平均分子量

四、気体・液体間の状態変化

1. 蒸発

液体の分子が表面から気体になる現象。どの温度でも起こる。

  • 気液平衡：蒸発と凝縮の速度が等しく，見かけ上，変化が起こらない状態。
  • (飽和) 蒸気圧：気液平衡にあるときの蒸気の圧力。

一定温度の蒸気圧は物質ごとに一定で，他の気体が共存しても変わらない。

蒸気圧は温度だけに依存し，高いほど大きい。

・蒸気圧曲線：液体の蒸気圧と温度との関係を示したグラフ。
